# Mini-Kernel Simulador
Um simulador de Kernel de Sistema Operacional desenvolvido em C, capaz de carregar programas, gerenciar memória, escalonar processos e simular um sistema de arquivos básico.

- Funcionalidades:

Gerenciamento de Processos (Load, Kill, ps)
Simulação de Memória (Alocação First-Fit com vetor global)
Interpretação de Instruções (malloc, wait, print)
File System Simulado (Create, Edit, Write, Delete)
